About

"APIs, a menu, but for applications"

APIs have become indispensible for developers wishing to communicate with other applications. As a beginner to the field, this can be rather daunting. The question we have:
Is there a simple way to get started?

The objective of this project is to build a simple API tester containing minimal functions for beginners to get started.
This project involves the following:
1. Background Research
2. Design Ideation
3. Coding the Application

Find out more!

Timeline

Our Project Journey

  • 1st February 2021

    Project commences!

    The imPoster developer team was formed. Research and brainstorming was done to come up with a product.

  • 8th February 2021

    User Stories

    The scope of the project was worked out and user stories were crafted to help set up the project direction.

  • 15th February 2021

    Design Ideation

    The features for the product were discussed and a simple user guide was drafted.

  • 22nd February 2021

    Project Setup

    Work for the initial documentation of the project was allocated and the features further discussed.

  • 2nd March 2021

    v1.1 milestone

    Milestone v1.1 was concluded as we wrapped up initial documentation and the discussion shifted onto the next milestone.

  • 10th March 2021

    Production Release!

    Milestone v1.2 was completed with basic functionalities including the sending of GET and POST requests!

  • 17th March 2021

    Second Public Release!

    Milestone v1.2b saw a second release with a minimal functioning version of all features included! Plenty of bug fixes were done too!

  • 24th March 2021

    Milestone v1.3 Complete!

    Milestone v1.3 saw heavy UI enhancements as well as the completion of all main functionalities!
